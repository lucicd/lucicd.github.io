Recognizing Arrays

Arrays are objects in JavaScript. The best way to recognize an array is to use Array.isArray method.

Using Array.isArray - case 1, real array

        var myArray = [1, 2, 3, 4, 5];
        console.log(Array.isArray(myArray));
This should show true.
