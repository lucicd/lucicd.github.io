Classical Inheritance

In classical inheritance, JavaScript is trying to imitate classical OO languages such as Java and C++.

Basic Inheritance Example

        //Parent class
        function Person(name, age) {
          this.name = name;
          this.age = age;
        }
        
        Person.prototype.greetings = function() {
            return 'Hello, my name is ' + this.name + '.';
        };

        //Child class
        function Student(name, age, major) {
          Person.call(this, name, age);
          this.major = major;
        }
        //Boiler plate code to set prototype and constructor
        Student.prototype = Object.create(Person.prototype);
        Student.prototype.constructor = Student;
        //Ehnancement of the child class
        Student.prototype.showMajor = function() {
          return 'My major is ' + this.major + '.';
        };
